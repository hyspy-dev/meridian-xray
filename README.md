# Meridian XRay

XRay module for the Meridian Proxy.

## Installation
Place the JAR in the world's module directory:
`%AppData%\Roaming\Hytale\UserData\Saves\<WorldName>\modules\`

### Internal Target List:
* **Soil_**: Grass, Dirt, etc.
* **Rock_**: Stone, Sandstone, Basalt, Volcanic, etc.

## Requirements
* `meridian-proxy`
